Radar Data Availability Figures

These are some useful figures showing airborne radar sounder data availability and survey density for the Antarctic and Greenland ice sheets, using data from xOPR and BedMap catalogs. Generated automatically on each push to main.

See the code to generate these figures on GitHub.

Combined Data Availability

uv run python plot_combined_availability.py

Survey Density: BedMap Antarctica

uv run python plot_survey_density.py --source bedmap_local --region antarctica

Survey Density: BedMap Antarctica (Amundsen Sea Embayment)

uv run python plot_survey_density.py --source bedmap_local --region antarctica --zoom ase

Survey Density: xOPR Greenland

uv run python plot_survey_density.py --source xopr --region greenland
